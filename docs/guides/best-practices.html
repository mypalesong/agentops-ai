Best Practices

Learn how to get the most out of AgentOps with these recommended practices.

Session Management​

Use Meaningful Tags​

# Good - descriptive and filterable
session = agentops.start_session(tags=[
    "production",
    "customer-support",
    "gpt-4",
    "v2.3.1"
])

# Bad - too generic
session = agentops.start_session(tags=["test"])

Always End Sessions​

try:
    session = agentops.start_session(tags=["task"])
    result = agent.run(task)
    session.end(end_state="Success")
except Exception as e:
    session.end(end_state="Fail", end_state_reason=str(e))
    raise

Provide End State Reasons​

session.end(
    end_state="Fail",
    end_state_reason="OpenAI API rate limit exceeded"
)

Event Recording​

Record Business-Critical Events​

agentops.record(Event(
    event_type="order_completed",
    params={
        "order_id": order.id,
        "value": order.total,
        "items": len(order.items)
    }
))

Use Consistent Event Types​

# Good - consistent naming
"user_login"
"user_logout"
"user_profile_updated"

# Bad - inconsistent
"UserLogin"
"user-logout"
"updateProfile"

Cost Optimization​

Use Appropriate Models​

def select_model(complexity):
    if complexity == "simple":
        return "gpt-3.5-turbo"
    elif complexity == "moderate":
        return "gpt-4-turbo"
    else:
        return "gpt-4"

Set Token Limits​

response = client.chat.completions.create(
    model="gpt-4",
    messages=messages,
    max_tokens=500  # Prevent runaway costs
)

Error Handling​

Capture Context​

try:
    result = process(data)
except Exception as e:
    agentops.record(ErrorEvent(
        error_type=type(e).__name__,
        details=str(e),
        params={
            "input_size": len(data),
            "step": current_step
        }
    ))
    raise

Security​

Never Log Sensitive Data​

# Good
agentops.record(Event(
    event_type="user_action",
    params={"user_id": user.id}
))

# Bad - contains PII
agentops.record(Event(
    event_type="user_action",
    params={"email": user.email}
))

Production Checklist​

  • Use environment variables for API keys
  • Set appropriate tags for filtering
  • Configure budget alerts
  • Handle errors gracefully
  • Avoid logging sensitive data
  • Review session end states
  • Monitor cost trends
