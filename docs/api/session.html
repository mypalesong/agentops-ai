Session API

The Session class represents a tracking session.

Class: Session​

class Session:
    session_id: str
    tags: List[str]
    config: dict
    is_running: bool

Properties​

PropertyTypeDescription
session_idstrUnique session identifier (UUID)
tagsList[str]Session tags
configdictSession configuration
is_runningboolWhether session is active

Methods​

session.record()​

Record an event to this session.

session.record(event: Event) -> None

Example​

session.record(Event(
    event_type="my_event",
    params={"key": "value"}
))

session.add_tags()​

Add tags to the session.

session.add_tags(tags: List[str]) -> None

Example​

session.add_tags(["new_tag", "another_tag"])

session.end()​

End the session.

session.end(
    end_state: str,
    end_state_reason: str = None
) -> None

Parameters​

ParameterTypeDefaultDescription
end_statestrRequired"Success", "Fail", or "Indeterminate"
end_state_reasonstrNoneExplanation

Example​

session.end(
    end_state="Success",
    end_state_reason="Completed successfully"
)

End States​

StateDescription
"Success"Session completed successfully
"Fail"Session failed
"Indeterminate"Outcome unclear

Usage Example​

import agentops

agentops.init(api_key="your-api-key", auto_start_session=False)

# Start session
session = agentops.start_session(tags=["example"])

# Access properties
print(f"Session ID: {session.session_id}")
print(f"Tags: {session.tags}")

# Record events
session.record(Event(event_type="start", params={}))

# Add more tags
session.add_tags(["additional"])

# End session
session.end(end_state="Success")
