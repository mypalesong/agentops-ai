Client API

The main AgentOps client API for initialization and session management.

agentops.init()​

Initialize the AgentOps SDK.

agentops.init(
    api_key: str = None,
    parent_key: str = None,
    endpoint: str = "https://api.agentops.ai",
    max_wait_time: int = 5000,
    max_queue_size: int = 100,
    default_tags: List[str] = None,
    instrument_llm_calls: bool = True,
    auto_start_session: bool = True,
    skip_auto_end_session: bool = False,
    env_data_opt_out: bool = False,
    inherited_session_id: str = None
) -> None

Parameters​

ParameterTypeDefaultDescription
api_keystrNoneAgentOps API key (or use env var)
parent_keystrNoneParent API key for organizations
endpointstrhttps://api.agentops.aiAPI endpoint URL
max_wait_timeint5000Max milliseconds before queue flush
max_queue_sizeint100Max events before queue flush
default_tagsList[str]NoneDefault tags for all sessions
instrument_llm_callsboolTrueAuto-instrument LLM providers
auto_start_sessionboolTrueStart session automatically
skip_auto_end_sessionboolFalseSkip auto session end on exit
env_data_opt_outboolFalseDon't collect environment data
inherited_session_idstrNoneContinue existing session

Example​

import agentops

agentops.init(
    api_key="your-api-key",
    default_tags=["production"],
    max_wait_time=10000
)

agentops.start_session()​

Start a new tracking session.

agentops.start_session(
    tags: List[str] = None,
    config: dict = None,
    inherited_session_id: str = None
) -> Session

Parameters​

ParameterTypeDefaultDescription
tagsList[str]NoneTags for this session
configdictNoneConfiguration metadata
inherited_session_idstrNoneContinue existing session

Returns​

Session object.

Example​

session = agentops.start_session(
    tags=["customer-support", "gpt-4"],
    config={"user_id": "123", "priority": "high"}
)

agentops.end_session()​

End the current session.

agentops.end_session(
    end_state: str,
    end_state_reason: str = None
) -> None

Parameters​

ParameterTypeDefaultDescription
end_statestrRequired"Success", "Fail", or "Indeterminate"
end_state_reasonstrNoneExplanation for end state

Example​

agentops.end_session(
    end_state="Success",
    end_state_reason="All tasks completed"
)

agentops.record()​

Record an event to the current session.

agentops.record(event: Event) -> None

Example​

from agentops import Event

agentops.record(Event(
    event_type="custom_event",
    params={"key": "value"}
))
