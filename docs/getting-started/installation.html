Installation

This guide covers how to install the AgentOps SDK and set up your development environment.

Requirements​

Before installing AgentOps, ensure you have:

  • Python 3.8+ - AgentOps requires Python 3.8 or higher
  • pip - Python package manager
  • API Key - Get your free API key from app.agentops.ai

Quick Install​

Install AgentOps using pip:

pip install agentops

For the latest development version:

pip install agentops --upgrade

Installation with Extras​

AgentOps provides optional extras for specific integrations:

# Install with LangChain support
pip install agentops[langchain]

# Install with all integrations
pip install agentops[all]

# Install development dependencies
pip install agentops[dev]

Environment Setup​

Setting Your API Key​

The recommended way to configure your API key is through environment variables:

Linux/macOS
export AGENTOPS_API_KEY="your-api-key-here"
Windows (PowerShell)
$env:AGENTOPS_API_KEY="your-api-key-here"
.env file
AGENTOPS_API_KEY=your-api-key-here

Using dotenv​

For local development, we recommend using python-dotenv:

pip install python-dotenv
config.py
from dotenv import load_dotenv
import os

# Load environment variables from .env file
load_dotenv()

# Access your API key
api_key = os.getenv("AGENTOPS_API_KEY")

Verifying Installation​

Verify your installation with this simple test:

verify_install.py
import agentops

# Check version
print(f"AgentOps version: {agentops.__version__}")

# Initialize and test connection
agentops.init(api_key="your-api-key")

# Create a test session
session = agentops.start_session(tags=["test", "installation"])

# Record a test event
agentops.record(agentops.Event(
    event_type="installation_test",
    params={"status": "success"}
))

# End session
agentops.end_session(end_state="Success")

print("Installation verified successfully!")

Run the verification:

python verify_install.py

Virtual Environment Setup​

We recommend using a virtual environment for your projects:

Using venv
# Create virtual environment
python -m venv agentops-env

# Activate (Linux/macOS)
source agentops-env/bin/activate

# Activate (Windows)
.\agentops-env\Scripts\activate

# Install AgentOps
pip install agentops
Using conda
# Create conda environment
conda create -n agentops python=3.11

# Activate
conda activate agentops

# Install AgentOps
pip install agentops

Docker Installation​

For containerized deployments:

Dockerfile
FROM python:3.11-slim

WORKDIR /app

# Install AgentOps
RUN pip install agentops

# Copy your application
COPY . .

# Set environment variables
ENV AGENTOPS_API_KEY=${AGENTOPS_API_KEY}

CMD ["python", "main.py"]
docker-compose.yml
version: '3.8'
services:
  agent:
    build: .
    environment:
      - AGENTOPS_API_KEY=${AGENTOPS_API_KEY}
    volumes:
      - .:/app

IDE Setup​

VS Code​

Install recommended extensions for the best development experience:

.vscode/extensions.json
{
  "recommendations": [
    "ms-python.python",
    "ms-python.vscode-pylance",
    "ms-python.black-formatter"
  ]
}

PyCharm​

  1. Go to File > Settings > Project > Python Interpreter
  2. Add a new interpreter or use existing virtual environment
  3. Install agentops package via the package manager

Troubleshooting​

Common Issues​

SSL Certificate Errors

If you encounter SSL errors, ensure your Python installation has proper certificates:

# macOS
pip install certifi
/Applications/Python\ 3.x/Install\ Certificates.command
Network Proxy

If you're behind a corporate proxy:

export HTTPS_PROXY=http://proxy.company.com:8080
export HTTP_PROXY=http://proxy.company.com:8080

Version Conflicts​

If you experience dependency conflicts:

# Check installed versions
pip list | grep -E "agentops|openai|langchain"

# Upgrade all related packages
pip install --upgrade agentops openai anthropic
