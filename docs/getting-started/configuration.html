Configuration

AgentOps provides flexible configuration options to customize its behavior for your specific use case. This guide covers all available configuration options.

Initialization Options​

The agentops.init() function accepts several configuration parameters:

import agentops

agentops.init(
    api_key="your-api-key",           # Required: Your AgentOps API key
    parent_key=None,                   # Optional: Parent API key for organizations
    endpoint="https://api.agentops.ai", # Optional: API endpoint
    max_wait_time=5000,                # Optional: Max time to wait for events (ms)
    max_queue_size=100,                # Optional: Max events in queue
    default_tags=["production"],       # Optional: Default tags for sessions
    instrument_llm_calls=True,         # Optional: Auto-instrument LLM calls
    auto_start_session=True,           # Optional: Auto-start session
    skip_auto_end_session=False,       # Optional: Skip auto-ending sessions
    env_data_opt_out=False,            # Optional: Opt out of environment data
)

Configuration Reference​

Core Settings​

ParameterTypeDefaultDescription
api_keystrRequiredYour AgentOps API key
parent_keystrNoneParent API key for multi-tenant setups
endpointstrhttps://api.agentops.aiAgentOps API endpoint

Session Settings​

ParameterTypeDefaultDescription
auto_start_sessionboolTrueAutomatically start a session on init
skip_auto_end_sessionboolFalseDon't auto-end session on program exit
default_tagslist[str][]Tags applied to all sessions
inherited_session_idstrNoneContinue from an existing session

Performance Settings​

ParameterTypeDefaultDescription
max_wait_timeint5000Maximum ms to wait for event queue
max_queue_sizeint100Maximum events in the queue

Instrumentation Settings​

ParameterTypeDefaultDescription
instrument_llm_callsboolTrueAuto-instrument LLM provider calls
env_data_opt_outboolFalseDon't collect environment metadata

Environment Variables​

Configure AgentOps using environment variables:

.env
# Required
AGENTOPS_API_KEY=your-api-key

# Optional
AGENTOPS_PARENT_KEY=parent-api-key
AGENTOPS_ENDPOINT=https://api.agentops.ai
AGENTOPS_MAX_WAIT_TIME=5000
AGENTOPS_MAX_QUEUE_SIZE=100
AGENTOPS_ENV_DATA_OPT_OUT=false
import agentops
import os

# API key can be read from environment automatically
agentops.init()  # Uses AGENTOPS_API_KEY env var

Configuration Patterns​

Development Configuration​

config/development.py
import agentops

def init_agentops_dev():
    agentops.init(
        api_key=os.getenv("AGENTOPS_API_KEY"),
        default_tags=["development", "debug"],
        max_wait_time=30000,  # Longer timeout for debugging
        max_queue_size=500,   # Larger queue for verbose logging
    )

Production Configuration​

config/production.py
import agentops

def init_agentops_prod():
    agentops.init(
        api_key=os.getenv("AGENTOPS_API_KEY"),
        default_tags=["production"],
        max_wait_time=5000,
        max_queue_size=100,
        skip_auto_end_session=False,  # Ensure sessions end properly
    )

Testing Configuration​

config/testing.py
import agentops

def init_agentops_test():
    agentops.init(
        api_key=os.getenv("AGENTOPS_TEST_API_KEY"),
        default_tags=["test", "ci"],
        auto_start_session=False,  # Manual control in tests
    )

Session Configuration​

Configure individual sessions with specific settings:

import agentops

agentops.init(api_key="your-api-key", auto_start_session=False)

# Start session with configuration
session = agentops.start_session(
    tags=["customer-support", "v2.1"],
    config={
        "user_id": "user-123",
        "environment": "production",
        "feature_flags": {
            "new_model": True,
            "streaming": False
        }
    },
    video=False  # Disable session replay video
)

Provider-Specific Configuration​

OpenAI​

import agentops
from openai import OpenAI

agentops.init(api_key="your-api-key")

# OpenAI client is automatically instrumented
client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY"),
    timeout=30.0,
    max_retries=3
)

Anthropic​

import agentops
from anthropic import Anthropic

agentops.init(api_key="your-api-key")

# Anthropic client is automatically instrumented
client = Anthropic(
    api_key=os.getenv("ANTHROPIC_API_KEY")
)

LangChain​

import agentops
from langchain_openai import ChatOpenAI

# Initialize AgentOps with LangChain handler
agentops.init(
    api_key="your-api-key",
    default_tags=["langchain"]
)

# Create LangChain components
llm = ChatOpenAI(model="gpt-4")

Logging Configuration​

Configure AgentOps logging level:

import logging
import agentops

# Set logging level
logging.getLogger("agentops").setLevel(logging.DEBUG)

# Or use environment variable
# AGENTOPS_LOG_LEVEL=DEBUG

agentops.init(api_key="your-api-key")

Log Levels​

LevelDescription
DEBUGDetailed diagnostic information
INFOGeneral operational messages
WARNINGWarning messages for potential issues
ERRORError messages for failures
CRITICALCritical failures

Network Configuration​

Proxy Settings​

import os

# Set proxy before initializing
os.environ["HTTPS_PROXY"] = "http://proxy.company.com:8080"
os.environ["HTTP_PROXY"] = "http://proxy.company.com:8080"

import agentops
agentops.init(api_key="your-api-key")

Custom Endpoint​

For self-hosted or enterprise deployments:

import agentops

agentops.init(
    api_key="your-api-key",
    endpoint="https://your-agentops-instance.company.com"
)

Timeout Configuration​

import agentops

agentops.init(
    api_key="your-api-key",
    max_wait_time=10000,  # 10 seconds
)

Privacy Configuration​

Opt-Out of Environment Data​

import agentops

agentops.init(
    api_key="your-api-key",
    env_data_opt_out=True  # Don't collect system info
)

Sensitive Data Handling​

import agentops

# Use tags to mark sensitive sessions
session = agentops.start_session(
    tags=["pii", "gdpr-compliant"]
)

# Avoid logging sensitive data in custom events
agentops.record(agentops.Event(
    event_type="user_action",
    params={
        "action": "login",
        "user_id": hash(user.email),  # Hash sensitive data
        # Don't include: password, email, phone, etc.
    }
))

Multi-Environment Configuration​

config.py
import os
import agentops

def get_agentops_config():
    env = os.getenv("APP_ENV", "development")

    configs = {
        "development": {
            "api_key": os.getenv("AGENTOPS_API_KEY_DEV"),
            "default_tags": ["development"],
            "max_wait_time": 30000,
        },
        "staging": {
            "api_key": os.getenv("AGENTOPS_API_KEY_STAGING"),
            "default_tags": ["staging"],
            "max_wait_time": 10000,
        },
        "production": {
            "api_key": os.getenv("AGENTOPS_API_KEY_PROD"),
            "default_tags": ["production"],
            "max_wait_time": 5000,
        }
    }

    return configs.get(env, configs["development"])

# Initialize with environment-specific config
config = get_agentops_config()
agentops.init(**config)
