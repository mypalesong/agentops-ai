Event Queue​

Events are queued and sent in batches for efficiency:

class EventQueue:
    """Manages event batching and transmission"""

    def __init__(self, max_size=100, max_wait_ms=5000):
        self.queue = []
        self.max_size = max_size
        self.max_wait_ms = max_wait_ms
        self.worker_thread = Thread(target=self._process_queue)

    def add(self, event):
        self.queue.append(event)
        if len(self.queue) >= self.max_size:
            self._flush()

    def _process_queue(self):
        while True:
            sleep(self.max_wait_ms / 1000)
            if self.queue:
                self._flush()

    def _flush(self):
        events = self.queue.copy()
        self.queue.clear()
        self._send_batch(events)

Data Models​

Session Schema​

interface Session {
  id: string;                    // Unique session ID (UUID)
  project_id: string;            // Project identifier
  start_timestamp: string;       // ISO 8601 timestamp
  end_timestamp?: string;        // ISO 8601 timestamp
  end_state?: "Success" | "Fail" | "Indeterminate";
  end_state_reason?: string;
  tags: string[];
  host_env: {
    os: string;
    python_version: string;
    sdk_version: string;
  };
  config: Record<string, any>;
}

Event Schema​

interface Event {
  id: string;                    // Unique event ID
  session_id: string;            // Parent session
  event_type: string;            // Type identifier
  timestamp: string;             // ISO 8601 timestamp
  params: Record<string, any>;   // Event-specific data
  result?: "success" | "fail" | "indeterminate";
}

interface LLMEvent extends Event {
  event_type: "llm";
  params: {
    provider: string;            // "openai", "anthropic", etc.
    model: string;               // "gpt-4", "claude-3", etc.
    prompt: Message[];           // Input messages
    completion: string;          // Output text
    prompt_tokens: number;
    completion_tokens: number;
    cost: number;
    latency_ms: number;
  };
}

Retry Logic​

class HTTPClient:
    def send(self, data, max_retries=3):
        for attempt in range(max_retries):
            try:
                response = self._post(data)
                if response.ok:
                    return response
            except Exception as e:
                if attempt == max_retries - 1:
                    raise
                sleep(2 ** attempt)  # Exponential backoff

Failure Handling​

  • Events buffered locally on network failure
  • Automatic retry with exponential backoff
  • Dead letter queue for failed events
  • Graceful degradation (agent continues working)
