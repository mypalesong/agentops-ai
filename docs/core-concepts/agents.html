Agents

AgentOps provides specialized tracking for AI agents, allowing you to monitor their lifecycle, actions, and interactions within your application.

What is an Agent?​

In AgentOps, an Agent is an autonomous entity that can:

  • Make decisions based on input
  • Execute actions and use tools
  • Interact with language models
  • Maintain state across interactions

Tracking Agents​

Using the @track_agent Decorator​

The simplest way to track an agent is with the @track_agent decorator:

from agentops import track_agent

@track_agent(name="ResearchAgent")
class ResearchAgent:
    def __init__(self, model="gpt-4"):
        self.model = model
        self.findings = []

    def research(self, topic: str) -> str:
        """Research a topic and return findings."""
        # Your research logic here
        return findings

    def summarize(self) -> str:
        """Summarize all findings."""
        return summary

Multi-Agent Systems​

AgentOps excels at tracking complex multi-agent architectures:

from agentops import track_agent

@track_agent(name="Coordinator")
class CoordinatorAgent:
    def __init__(self):
        self.workers = []

    def add_worker(self, worker):
        self.workers.append(worker)

    def delegate(self, task):
        # Delegate to appropriate worker
        worker = self.select_worker(task)
        return worker.execute(task)

@track_agent(name="ResearchWorker")
class ResearchWorker:
    def execute(self, task):
        # Research execution
        pass

@track_agent(name="WritingWorker")
class WritingWorker:
    def execute(self, task):
        # Writing execution
        pass

# Usage
coordinator = CoordinatorAgent()
coordinator.add_worker(ResearchWorker())
coordinator.add_worker(WritingWorker())

# All agent interactions are tracked
result = coordinator.delegate("Write a report on AI trends")

Agent State Tracking​

Track agent state changes:

from agentops import track_agent, Event

@track_agent(name="StatefulAgent")
class StatefulAgent:
    def __init__(self):
        self.state = "idle"
        self._record_state_change("initialized")

    def _record_state_change(self, new_state):
        old_state = self.state
        self.state = new_state
        agentops.record(Event(
            event_type="agent_state_change",
            params={
                "agent": self.__class__.__name__,
                "old_state": old_state,
                "new_state": new_state
            }
        ))

    def start_task(self, task):
        self._record_state_change("working")
        # Process task
        self._record_state_change("idle")

Framework Integrations​

LangChain Agents​

import agentops
from langchain.agents import initialize_agent, Tool
from langchain_openai import ChatOpenAI

agentops.init(api_key="your-api-key")

# LangChain agents are automatically tracked
tools = [
    Tool(
        name="Search",
        func=search_function,
        description="Search for information"
    )
]

llm = ChatOpenAI(model="gpt-4")
agent = initialize_agent(tools, llm, agent="zero-shot-react-description")

# Agent execution is tracked
result = agent.run("Find the latest news about AI")

Custom Agent Implementation​

For custom agent frameworks, implement manual tracking:

import agentops
from agentops import track_agent, Event

@track_agent(name="CustomAgent")
class CustomAgent:
    def __init__(self):
        self.agent_id = None  # Set by decorator

    def think(self, input_data):
        """Agent reasoning step."""
        agentops.record(Event(
            event_type="agent_think",
            params={
                "agent_id": self.agent_id,
                "input_length": len(str(input_data))
            }
        ))
        # Reasoning logic
        return decision

    def act(self, decision):
        """Agent action step."""
        agentops.record(Event(
            event_type="agent_act",
            params={
                "agent_id": self.agent_id,
                "action": decision.action,
                "confidence": decision.confidence
            }
        ))
        # Action execution
        return result

    def run(self, task):
        """Main agent loop."""
        while not self.is_complete(task):
            decision = self.think(task.current_state)
            result = self.act(decision)
            task.update(result)
        return task.result

Agent Communication​

Track inter-agent communication:

from agentops import Event

def send_message(sender, receiver, message):
    """Send a message between agents."""
    agentops.record(Event(
        event_type="agent_message",
        params={
            "sender": sender.name,
            "receiver": receiver.name,
            "message_type": message.type,
            "message_length": len(message.content)
        }
    ))
    receiver.receive(message)

Best Practices​

1. Name Agents Meaningfully​

# Good - descriptive names
@track_agent(name="CustomerSupportAgent")
@track_agent(name="DataAnalysisAgent")
@track_agent(name="ContentGenerationAgent")

# Bad - generic names
@track_agent(name="Agent1")
@track_agent(name="MyAgent")

2. Track Agent Decisions​

@track_agent(name="DecisionAgent")
class DecisionAgent:
    def decide(self, options):
        selected = self._evaluate(options)

        agentops.record(Event(
            event_type="agent_decision",
            params={
                "options": [o.name for o in options],
                "selected": selected.name,
                "reasoning": self.last_reasoning
            }
        ))

        return selected

3. Handle Agent Errors​

@track_agent(name="RobustAgent")
class RobustAgent:
    def execute(self, task):
        try:
            result = self._process(task)
            return result
        except Exception as e:
            agentops.record(ErrorEvent(
                error_type=type(e).__name__,
                details=str(e),
                params={"agent": self.__class__.__name__, "task": task.id}
            ))
            raise

4. Monitor Agent Performance​

import time

@track_agent(name="PerformanceAgent")
class PerformanceAgent:
    def run(self, task):
        start = time.time()

        result = self._execute(task)

        duration = time.time() - start
        agentops.record(Event(
            event_type="agent_performance",
            params={
                "agent": self.__class__.__name__,
                "task_type": task.type,
                "duration_seconds": duration,
                "success": result.success
            }
        ))

        return result
