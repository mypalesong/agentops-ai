Multi-Agent Systems

AgentOps excels at tracking complex multi-agent systems, providing visibility into agent interactions, delegation patterns, and collaborative workflows.

Architecture Patterns​

Coordinator Pattern​

from agentops import track_agent

@track_agent(name="Coordinator")
class CoordinatorAgent:
    def __init__(self):
        self.workers = []

    def add_worker(self, worker):
        self.workers.append(worker)

    def delegate(self, task):
        worker = self.select_worker(task)
        return worker.execute(task)

@track_agent(name="Worker")
class WorkerAgent:
    def execute(self, task):
        # Task execution tracked
        pass

Pipeline Pattern​

@track_agent(name="Stage1")
class PreprocessingAgent:
    def process(self, data):
        return cleaned_data

@track_agent(name="Stage2")
class AnalysisAgent:
    def analyze(self, data):
        return analysis

@track_agent(name="Stage3")
class OutputAgent:
    def format(self, analysis):
        return formatted_output

# Pipeline execution - all stages tracked
def run_pipeline(data):
    stage1 = PreprocessingAgent()
    stage2 = AnalysisAgent()
    stage3 = OutputAgent()

    result = stage1.process(data)
    result = stage2.analyze(result)
    return stage3.format(result)

Inter-Agent Communication​

def send_message(sender, receiver, message):
    agentops.record(Event(
        event_type="agent_message",
        params={
            "sender": sender.name,
            "receiver": receiver.name,
            "message_type": message.type
        }
    ))
    receiver.receive(message)

Tracking Workflows​

workflow_id = str(uuid.uuid4())

agentops.record(Event(
    event_type="workflow_start",
    params={"workflow_id": workflow_id, "agents": ["A", "B", "C"]}
))

# Track each agent's contribution
for agent in agents:
    agentops.record(Event(
        event_type="agent_step",
        params={
            "workflow_id": workflow_id,
            "agent": agent.name,
            "action": agent.last_action
        }
    ))

agentops.record(Event(
    event_type="workflow_end",
    params={"workflow_id": workflow_id, "success": True}
))

Best Practices​

  1. Name agents clearly: Use descriptive, unique names
  2. Track handoffs: Log when work transfers between agents
  3. Correlate events: Use workflow IDs to group related events
  4. Monitor bottlenecks: Track which agents slow down pipelines
