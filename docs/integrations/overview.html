How Integration Works​

Auto-Instrumentation​

Most providers are automatically instrumented when you initialize AgentOps:

import agentops
from openai import OpenAI

# Initialize AgentOps - auto-instruments OpenAI
agentops.init(api_key="your-api-key")

# Use OpenAI normally - all calls are tracked
client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello!"}]
)

What Gets Captured​

For each LLM call, AgentOps captures:

  • Request details: Model, messages, parameters
  • Response: Content, finish reason
  • Tokens: Input and output counts
  • Timing: Latency measurements
  • Cost: Calculated from pricing data
  • Errors: Any exceptions or API errors

import agentops

Anthropic​

import agentops
from anthropic import Anthropic

agentops.init(api_key="your-api-key")

Integration Levels​

Level 1: Basic (Auto-Instrumented)​

Just initialize AgentOps and use your providers normally:

import agentops
agentops.init(api_key="your-api-key")

# All supported providers now tracked automatically

What you get:

  • LLM call logging
  • Token counting
  • Cost tracking
  • Basic error capture

Level 2: Enhanced (With Decorators)​

Add decorators for tool and agent tracking:

import agentops
from agentops import track_agent, tool

agentops.init(api_key="your-api-key")

@track_agent(name="MyAgent")
class MyAgent:
    @tool(name="search")
    def search(self, query):
        # Tool calls tracked
        pass

Additional features:

  • Tool execution tracking
  • Agent lifecycle events
  • Custom event correlation

Level 3: Full (Custom Events)​

Add custom events for complete observability:

import agentops
from agentops import Event

agentops.init(api_key="your-api-key")

# Log custom business events
agentops.record(Event(
    event_type="user_feedback",
    params={"rating": 5, "comment": "Great!"}
))

Complete feature set:

  • Business event tracking
  • Custom metrics
  • Advanced analytics

Common Patterns​

Multiple Providers​

Use multiple LLM providers in one session:

import agentops
from openai import OpenAI
from anthropic import Anthropic

agentops.init(api_key="your-api-key")

openai_client = OpenAI()
anthropic_client = Anthropic()

# Both are tracked in the same session
gpt_response = openai_client.chat.completions.create(...)
claude_response = anthropic_client.messages.create(...)

Async Support​

All integrations support async operations:

import agentops
import asyncio
from openai import AsyncOpenAI

agentops.init(api_key="your-api-key")

async def main():
    client = AsyncOpenAI()

    # Async calls tracked
    response = await client.chat.completions.create(
        model="gpt-4",
        messages=[{"role": "user", "content": "Hello!"}]
    )

asyncio.run(main())

Streaming​

Streaming responses are fully supported:

import agentops
from openai import OpenAI

agentops.init(api_key="your-api-key")
client = OpenAI()

# Streaming tracked - event recorded after stream completes
stream = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Tell me a story"}],
    stream=True
)

for chunk in stream:
    print(chunk.choices[0].delta.content or "", end="")

Disabling Auto-Instrumentation​

If needed, disable specific instrumentations:

import agentops

agentops.init(
    api_key="your-api-key",
    instrument_llm_calls=False  # Disable all auto-instrumentation
)

Verifying Integration​

Check that integration is working:

import agentops

agentops.init(api_key="your-api-key")

# Make a test call
from openai import OpenAI
client = OpenAI()
response = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "Test"}]
)

# Check the dashboard for the recorded event
agentops.end_session(end_state="Success")
print("Check dashboard for test session")
