Cost Tracking

AgentOps provides comprehensive cost tracking to help you monitor, analyze, and optimize your LLM API spending.

Automatic Cost Calculation​

AgentOps automatically calculates costs for all LLM calls based on current provider pricing:

import agentops
from openai import OpenAI

agentops.init(api_key="your-api-key")
client = OpenAI()

# Cost is automatically calculated and tracked
response = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Hello!"}]
)

# View in dashboard:
# - Model: gpt-4
# - Input tokens: 10
# - Output tokens: 25
# - Cost: $0.00125

Pricing Calculation​

# Example pricing calculation
pricing = {
    "gpt-4": {
        "input": 0.03,   # per 1K tokens
        "output": 0.06   # per 1K tokens
    },
    "gpt-4-turbo": {
        "input": 0.01,
        "output": 0.03
    },
    "gpt-3.5-turbo": {
        "input": 0.0005,
        "output": 0.0015
    }
}

# Cost = (input_tokens * input_price + output_tokens * output_price) / 1000
# For gpt-4 with 1000 input and 500 output tokens:
# Cost = (1000 * 0.03 + 500 * 0.06) / 1000 = $0.06

      - threshold: 75%

Budget Alerts​

# Programmatic budget monitoring
from agentops import Client

client = Client(api_key="your-api-key")

# Get current budget status
budget = client.get_budget_status()
print(f"Monthly budget: ${budget.limit}")
print(f"Current spend: ${budget.current}")
print(f"Remaining: ${budget.remaining}")
print(f"Percentage used: {budget.percentage_used}%")

# Set up alert webhook
client.set_budget_alert(
    threshold=0.9,  # 90%
    webhook_url="https://your-webhook.com/alerts"
)

Cost Optimization​

Identify Expensive Sessions​

# Find most expensive sessions
expensive_sessions = client.get_sessions(
    sort_by="cost",
    order="desc",
    limit=10
)

for session in expensive_sessions:
    print(f"Session {session.session_id}: ${session.total_cost:.2f}")
    print(f"  Model: {session.primary_model}")
    print(f"  Tokens: {session.total_tokens}")

Model Comparison​

# Compare model costs for similar tasks
comparison = client.compare_model_costs(

    models=["gpt-4", "gpt-4-turbo", "gpt-3.5-turbo"]

Cost Reduction Strategies​

1. Use Appropriate Models​

# Route simple queries to cheaper models
def select_model(query_complexity):
    if query_complexity == "simple":
        return "gpt-3.5-turbo"  # $0.002/1K tokens
    elif query_complexity == "moderate":
        return "gpt-4-turbo"    # $0.02/1K tokens
    else:
        return "gpt-4"          # $0.06/1K tokens

2. Optimize Prompts​

# Track prompt length vs cost
agentops.record(Event(
    event_type="prompt_optimization",
    params={
        "original_length": len(original_prompt),
        "optimized_length": len(optimized_prompt),
        "savings_percent": (1 - len(optimized_prompt)/len(original_prompt)) * 100
    }
))

3. Implement Caching​

# Cache common responses
from functools import lru_cache

@lru_cache(maxsize=1000)
def get_cached_response(query_hash):
    # Check cache before calling LLM
    pass

# Track cache hit rates
agentops.record(Event(
    event_type="cache_stats",
    params={
        "hits": cache_hits,
        "misses": cache_misses,
        "savings": estimated_savings
    }
))

4. Set Token Limits​

# Prevent runaway costs
response = client.chat.completions.create(
    model="gpt-4",
    messages=messages,
    max_tokens=500  # Limit output
)

# Track truncations
if response.choices[0].finish_reason == "length":
    agentops.record(Event(
        event_type="token_limit_hit",
        params={"max_tokens": 500}
    ))

Best Practices​

1. Set Realistic Budgets​

# Calculate budget based on expected usage
expected_sessions_per_day = 1000
avg_cost_per_session = 0.10
buffer = 1.2  # 20% buffer

daily_budget = expected_sessions_per_day * avg_cost_per_session * buffer
monthly_budget = daily_budget * 30

2. Monitor Cost Anomalies​

# Alert on unusual spending
client.set_anomaly_detection(
    metric="hourly_cost",
    sensitivity="medium",
    channels=["slack"]
)

3. Regular Cost Reviews​

  • Daily: Check for anomalies
  • Weekly: Review trends and efficiency
  • Monthly: Full cost analysis and optimization

4. Track ROI​

# Link costs to business value
agentops.record(Event(
    event_type="business_outcome",
    params={
        "session_id": session.session_id,
        "session_cost": session.total_cost,
        "value_generated": 50.00,  # e.g., ticket resolved
        "roi": (50.00 - session.total_cost) / session.total_cost
    }
))
