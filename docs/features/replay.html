Session Replay

Session Replay allows you to step through any agent session event by event, providing complete visibility into agent behavior.

Overview​

Session Replay captures:

  • Every LLM call with full prompts and responses
  • Tool invocations and results
  • Custom events and state changes
  • Timing information between events
  • Errors and their context

Using Session Replay​

Access from Dashboard​

  1. Navigate to Sessions in the AgentOps dashboard
  2. Click on any session to open it
  3. Click Replay to enter replay mode

Replay Controls​

ControlAction
PlayAuto-advance through events
PauseStop auto-advance
Step ForwardMove to next event
Step BackMove to previous event
Jump to StartGo to first event
Jump to EndGo to last event
SpeedAdjust playback speed (0.5x - 4x)

Export Replay​

Export session replay for sharing:

from agentops import Client

client = Client(api_key="your-api-key")

# Export replay data
replay_data = client.export_replay(
    session_id="sess_abc123",
    format="json"
)

# Save to file
with open("session_replay.json", "w") as f:
    json.dump(replay_data, f, indent=2)

Programmatic Replay​

Fetch Session Events​

from agentops import Client

client = Client(api_key="your-api-key")

# Get session with all events
session = client.get_session("sess_abc123", include_events=True)

# Iterate through events
for event in session.events:
    print(f"[{event.timestamp}] {event.event_type}")
    print(f"  Params: {event.params}")
    print()

Replay in Code​

# Recreate session flow for testing
def replay_session(session_id):
    session = client.get_session(session_id, include_events=True)

    for event in session.events:
        if event.event_type == "llm":
            print(f"LLM Call ({event.params['model']}):")
            print(f"  Input: {event.params['prompt']}")
            print(f"  Output: {event.params['completion']}")

        elif event.event_type == "tool":
            print(f"Tool Call ({event.params['tool_name']}):")
            print(f"  Input: {event.params['input']}")
            print(f"  Output: {event.params['output']}")

        elif event.event_type == "error":
            print(f"Error: {event.params['error_type']}")
            print(f"  Details: {event.params['details']}")

Replay for Debugging​

def debug_failed_session(session_id):
    """Analyze a failed session step by step."""
    session = client.get_session(session_id, include_events=True)

    print(f"Session: {session.session_id}")
    print(f"End State: {session.end_state}")
    print(f"End Reason: {session.end_state_reason}")
    print("\n--- Event Replay ---\n")

    for i, event in enumerate(session.events):
        print(f"Step {i + 1}: {event.event_type}")
        print(f"Time: {event.timestamp}")

        if event.event_type == "error":
            print("⚠️  ERROR DETECTED")
            print(f"Type: {event.params.get('error_type')}")
            print(f"Details: {event.params.get('details')}")
            print("\n--- Events Before Error ---")

            # Show previous events for context
            for j in range(max(0, i - 3), i):
                prev = session.events[j]
                print(f"  [{prev.event_type}] {prev.params}")

        print()

Best Practices​

1. Add Context Events​

Make replay more useful by adding context:

# Before making decisions, log the reasoning
agentops.record(Event(
    event_type="agent_reasoning",
    params={
        "input": user_query,
        "options_considered": ["search", "calculate", "respond"],
        "selected": "search",
        "reasoning": "User asked about current events"
    }
))

2. Log State Changes​

Track important state transitions:

class StatefulAgent:
    def set_state(self, new_state):
        old_state = self.state
        self.state = new_state

        agentops.record(Event(
            event_type="state_change",
            params={
                "from": old_state,
                "to": new_state,
                "reason": self.last_action
            }
        ))

3. Include Input/Output​

Always include inputs and outputs for replay clarity:

@tool(name="process_data")
def process_data(data: dict) -> dict:
    agentops.record(Event(
        event_type="tool_input",
        params={"tool": "process_data", "input": data}
    ))

    result = do_processing(data)

    agentops.record(Event(
        event_type="tool_output",
        params={"tool": "process_data", "output": result}
    ))

    return result

4. Mark Important Events​

Highlight key moments in the session:

# Mark important decisions
agentops.record(Event(
    event_type="important_decision",
    params={
        "description": "Escalating to human support",
        "reason": "Complex technical issue detected",
        "confidence": 0.95
    }
))
