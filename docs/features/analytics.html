Success Rate Trends​

# Query success rates programmatically
from agentops import Client

client = Client(api_key="your-api-key")

metrics = client.get_analytics(
    metric="success_rate",
    period="7d",
    group_by="day"
)

for day in metrics:
    print(f"{day['date']}: {day['success_rate']:.1f}%")

Custom Analytics​

Creating Custom Metrics​

Define custom metrics based on your events:

# Track custom conversion metric
agentops.record(Event(
    event_type="conversion",
    params={
        "type": "signup",
        "source": "chatbot",
        "value": 1
    }
))

# Query custom metrics
custom_metrics = client.get_analytics(
    event_type="conversion",
    period="30d",
    aggregation="sum",
    group_by="params.type"
)

Funnel Analysis​

Track user journeys through your agent:

# Track funnel steps
funnel_events = [
    ("funnel_start", {"step": "greeting"}),
    ("funnel_step", {"step": "need_identification"}),
    ("funnel_step", {"step": "solution_offered"}),
    ("funnel_complete", {"step": "resolution"})
]

for event_type, params in funnel_events:
    agentops.record(Event(
        event_type=event_type,
        params={**params, "session_id": session.session_id}
    ))

Analytics API​

Query Analytics​

from agentops import Client

client = Client(api_key="your-api-key")

# Get session analytics
session_analytics = client.get_analytics(
    metric="sessions",
    period="30d",
    group_by="day",
    filters={
        "tags": ["production"],
        "end_state": "Success"
    }
)

# Get LLM analytics
llm_analytics = client.get_analytics(
    metric="llm_calls",
    period="7d",
    group_by="model",
    filters={
        "min_cost": 0.01
    }
)

# Get cost analytics
cost_analytics = client.get_analytics(
    metric="cost",
    period="30d",
    group_by=["model", "tag"],
    aggregation="sum"
)

Export Analytics​

# Export to CSV
analytics_data = client.export_analytics(
    metrics=["sessions", "cost", "tokens"],
    period="90d",
    format="csv"
)

with open("analytics_export.csv", "w") as f:
    f.write(analytics_data)

# Export to JSON
json_data = client.export_analytics(
    metrics=["sessions", "cost", "tokens"],
    period="90d",
    format="json"
)

Alerts Based on Analytics​

Anomaly Detection​

# Alert on anomalies
alerts:
  - name: "Unusual Error Rate"
    type: anomaly
    metric: error_rate
    sensitivity: high
    channels: [slack]

  - name: "Cost Spike"
    type: threshold
    metric: hourly_cost
    threshold: ">$50"
    channels: [email, slack]

Scheduled Reports​

# Weekly analytics report
reports:
  - name: "Weekly Performance"
    schedule: "0 9 * * MON"  # Every Monday at 9 AM
    metrics:
      - sessions
      - success_rate
      - avg_latency
      - total_cost
    compare_to: "previous_week"
    recipients: [team@company.com]

Best Practices​

1. Set Up Key Dashboards​

Create focused dashboards for different stakeholders:

  • Engineering: Error rates, latency, system health
  • Product: Usage patterns, feature adoption
  • Finance: Cost tracking, budget utilization

2. Track Business Metrics​

# Link technical metrics to business outcomes
agentops.record(Event(
    event_type="business_metric",
    params={
        "metric": "customer_satisfaction",
        "score": 4.5,
        "session_id": session.session_id
    }
))

3. Regular Review Cadence​

  • Daily: Check error rates and anomalies
  • Weekly: Review performance trends
  • Monthly: Analyze cost efficiency and ROI

4. Use Cohort Analysis​

Compare performance across different user segments:

# Tag sessions by cohort
session = agentops.start_session(tags=[
    f"cohort:{user.signup_month}",
    f"plan:{user.subscription_tier}",
    f"region:{user.region}"
])
